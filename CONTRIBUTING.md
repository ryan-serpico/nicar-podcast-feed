# Contributing

Thanks for your interest in contributing to the NICAR Sessions Podcast feed!

## Getting started

1. Fork the repo and clone it locally
2. Run `python3 generate_feed.py` to generate the feed (requires Python 3.9+, no external dependencies)
3. The output goes to `docs/feed.xml`

## Ways to contribute

- **Add past conferences**: If you have access to audio URLs from previous NICAR conferences, open an issue or PR to add them
- **Bug fixes**: If you notice incorrect episode data, broken links, or feed formatting issues, please open an issue
- **Improvements**: Ideas for better show notes, feed metadata, or the landing page are welcome

## Submitting changes

1. Create a branch for your changes
2. Run `python3 generate_feed.py` locally to make sure the feed generates without errors
3. Open a pull request with a clear description of what you changed and why

## Code style

- Keep it simple — the project intentionally uses only Python standard library
- Follow existing patterns in the codebase

## Questions?

Open an issue and we'll figure it out together.
